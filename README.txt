Poderosa

A terminal emulator for windows.

Supports:
 - SSH
   - Agent forwarding
   - X11 forwarding
   - Port forwarding (provided as a standalone tool)
 - SFTP, SCP
 - Telnet
 - XModem, ZModem
 - Serial port
 - Pipe (named pipe or a process)
